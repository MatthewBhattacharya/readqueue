Connect to Google Drive

ReadQueue watches a Google Drive folder and gives you a clean reading queue with offline support. Here's how to set it up:

1

Create a Drive folder

Create a folder in Google Drive called "ReadQueue" (or whatever you like). Drop your PDFs in there from any device.

2

Get a Google API Key

Go to console.cloud.google.com, create a project, enable the Google Drive API, and create an API key. Restrict it to the Drive API for safety.

3

Share folder or make it accessible

Either make the folder link-accessible ("Anyone with the link can view") or use OAuth. For simplicity, we'll use a shared link + API key approach. Paste the folder ID below — it's the long string in the folder URL after /folders/.
